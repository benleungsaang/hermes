## 1. 组合秤

- **适用**：颗粒 / 片状 / 块状等散装物料的多头称重（糖果、坚果、薯片、冻干、宠物零食等）
- **头数规格**：10 头 / 14 头（标准主力）
- **特殊类型**：
  - 防碎组合秤（缓落料、衬软）
  - 粘性物料组合秤（防粘涂层、防堵斗）
  - 超大 / 超小物料秤（不同斗容积）
- **价格档**：⏳ 待 B 补充
- **备注**：⏳

## 2. 螺杆秤

- **适用**：粉末 / 细小颗粒 / 流动性差的物料（面粉、奶粉、调料粉、咖啡粉、辣椒粉等）
- **螺杆规格**：⏳ 待 B 补充（按螺杆直径 / 螺距分）
- **价格档**：⏳
- **备注**：⏳

## 3. 量杯

- **适用**：颗粒 / 流动性好的物料（体积法计量，精度要求不高的小规格包装）
- **规格**：⏳ 按容积（ml/cc）
- **价格档**：⏳
- **备注**：⏳

## 4. 净重电子秤

- **适用**：单件物料或大规格整袋物料的称重（整只面包、整袋大米、整盒月饼等）
- **价格档**：⏳
- **备注**：⏳

## 5. 计量液体泵

- **适用**：液体 / 酱料 / 膏体（食用油、酱油、咖喱酱、番茄酱、蜂蜜、洗手液等）
- **泵类型**：⏳ 柱塞泵 / 蠕动泵 / 齿轮泵 等
- **价格档**：⏳
- **备注**：⏳

## 6. 提升机

- **适用**：把物料从低位提升到包装机进料口的连续输送
- **类型**：
  - Z 型提升机（适用多种物料，主流）
  - C 型提升机（节省空间）
- **载重规格**：⏳ 按 kg / 小时分档
- **价格档**：⏳
- **备注**：⏳

## 7. 袋模块（开底袋 / 开拉链 / 封拉链）

- **适用**：给袋式包装机（premade pouch）配套
- **模块类型**：
  - 开底袋模块
  - 开拉链模块
  - 封拉链模块
- **价格档**：⏳
- **备注**：⏳

## 8. 真空 / 充氮气 / 排空气 模块

- **适用**：需要控氧包装的食品（肉制品、坚果、咖啡、薯片等防氧化 / 保鲜）
- **模块类型**：
  - 抽真空
  - 充氮气（N2 置换空气）
  - 排空气（轻量版）
- **价格档**：⏳
- **备注**：⏳

## 9. 打码机 / 视觉检测

- **适用**：产品追溯、外观质量控制
- **打码机类型**：⏳ 喷码 / 激光 / 热转印 等
- **视觉检测**：⏳ 缺陷检测 / 字符识别 / 称重复核
- **价格档**：⏳
- **备注**：⏳

## 10. 输送机 / 料仓

- **适用**：物料缓冲、产线连接
- **输送机类型**：⏳ 皮带 / 链板 / 辊筒 等
- **料仓**：⏳ 按容积（L）分档

## 更新约定

- B 补充一条信息 → Hermes 整理成一行（不超 100 tokens）→ 追加到对应配件小节
- 价格档用区间表示（如 `¥2-5 万`），不写精确到个位的数字（销售报价有谈判空间）
- 范围/适用用一段话 + 关键参数，不要长篇
- 严重情况（新配件类型 / 价格剧烈变动）→ 标 ⚠️ 写到 `.learnings/domains/packaging-machines.md` 主文件
