## 一、型号编码规则

格式：`[结构]-[技术类型]-[结构分支(选填)]-[主要技术参数]-[技术特性标注(选填)]-[切刀方式(选填)]`

### 结构（首段）
| 编码 | 含义 | 类型 |
|---|---|---|
| HP | Horizontal Package | 枕式包装机 |
| VP | Vertical Package | 立式包装机 |

### 技术类型（第二段，必填）
| 编码 | 含义 | 说明 |
|---|---|---|
| BF | Bag Filling | 给袋（装袋、填袋）—— 预制袋 |
| BS | Bag Sleeving | 套袋（外袋套入、缩口封口） |
| FB | Film (to) Bag | 膜袋（卷膜制袋） |

### 结构分支（第三段，选填）
| 编码 | 含义 |
|---|---|
| BF | Bottom Film Loader —— 下走膜 |
| （无） | UF = Upper Film Loader 上走膜（默认不标注） |

### 主要技术参数（数字段）
最大膜宽 / 最高速度 / 最大载重 等

### 技术特性标注（选填，无 - 分隔，紧跟数字后或括号内）
| 编码 | 含义 |
|---|---|
| V | Vacuum 真空 |
| H | High Speed 高速型 |
| S / S(hr) | (Heat) Shrink 热收缩 |
| Oe | On Edge 无托机 |
| U | Unitized 一体机 |
| UD | 升级款 |
| ES | 电子秤系列 |
| E / T | 切刀变体 |
| (B) | 大轮径 / Big |

### 切刀方式（末尾，选填，仅枕式机相关）
| 编码 | 含义 |
|---|---|
| B | Box Motion 往复式 |
| （无） | R = Rotary Motion 旋转刀（默认不标注） |

### 最长示例
`HP-FB-BF-450S-B` → 枕式 + 膜袋 + 下走膜 + 膜宽450mm + 热收缩 + 往复式切刀

## 五、选型决策树（来自 .md 文档 + B 培训补充）

```
1. 大类（按物料 + 形态）：
   - 散装粉末/液体/颗粒 → 立式 VFFS 或 给袋机 都行（看袋型选择）
   - 定型单品 → 枕式 HFFS（卷膜枕式）
   - 大面包等大件 + 预制袋装不下 → 套袋（横式给袋，先撑袋再推入物料）

2. 袋型决定机型：
   - bag（卷膜袋）→ 立式机 VFFS（膜袋制袋）
   - pouch / premade bag / stand up bag（预制袋）→ 给袋机 Premade Bag
   - 枕式袋 → 枕式机 HFFS
   立式机与给袋机物料兼容范围完全一致（粉末/液体/颗粒都行），
   区别只在袋型。一般 pouch + 高展示效果会推荐给袋机更显高档

2.5 易碎性修正（2026-06-17 补充）：
   立式机和给袋机物料基本一样，但**落料高度不同**——
   立式机落料高度更高 → 易碎产品跌落破碎风险更大
   给袋机落料高度更低 → 对易碎产品更友好
   → 易碎颗粒（膨化食品、薯片、饼干、巧克力制品等）选型时应优先给袋机

3. 立式机/枕式机：按膜宽选型
   膜宽 = (成品袋宽 + 10~15mm) × 2
   ⚠️ 中封余量不应低于 10mm；接近极限膜宽时与工程师/客户沟通是
   选更大机型还是缩小袋宽

4. 枕式机额外确认：
   - 走膜方式（上走膜快 / 下走膜平稳）
   - 切刀类型（旋转最快/限矮产品 / 往复高开口/首选 / 上下切便宜/必停机）

5. 给袋机：按袋宽 + 工位
   6 工位 = 标准（拉链、开底袋等基础增值功能也能在 6 工位完成）
   8+ 工位 = 真空/充气/双落料等复杂高配

6. 计量选型（关键修正）：
   - 粉末 → 螺杆秤（性价比高）/ 量杯（仅常规干燥粉末）
   - 易吸潮 / 压缩比易变物料 → ❌ 量杯 ❌ 螺杆秤（都是体积计量，
     密度变化导致重量偏差）→ 改用净重式电子秤（称重反馈补偿）
   - 易碎颗粒 / 高精度需求 → 多头组合秤（防碎定制版）
   - 液体/膏体 → 计量液体泵
   - 选型无默认机型，以客户需求 / 物料价值 / 袋型成本为导向

7. 物料装填判断：
   - 客户手工测试能装 ≠ 机器能装
   - 手工有压实/抖动/扶正/调整位姿等操作空间
   - 机器预计装到 70-80% 满时要预警（让客户重新评估袋型）
```

## 六、命名规则补充（来自 B 培训）
- HP = Horizontal Package（枕式）
- VP = Vertical Package（立式）
- BF = Bag Filling（给袋/装袋填袋，预制袋）
- BS = Bag Sleeving（套袋，外袋套入缩口封口）
- FB = Film (to) Bag（膜袋，卷膜制袋）
- BF（第三段）= Bottom Film Loader 下走膜；UF（上走膜）默认不标
- 数字段：膜宽 / 速度 / 载重 等主要参数
- 特性标注（无 - 分隔）：V=真空 / H=高速 / S/S(hr)=热收缩 / Oe=无托机 /
  U=一体机 / UD=升级 / ES=电子秤系列 / E,T=切刀变体 / (B)=大轮径
- 切刀（仅枕式）：B=往复式；R=旋转式默认不标
- 历史错误：FTP = FB 的拼写错误

## 七、枕式机选型补充（B 培训）

- 送料方式：易滚动/易倒产品应用**推块输送带**（非纯皮带），保持产品位姿稳定
- 无托机 Oe：公司未采用，成本高，不在方案推荐范围内
- 旋转切刀高产品：通用做法看旋转刀的两个中心距能否容下产品，容不下只能用往复式或上下切，没有专门"高产品+高速"方案
- 吸嘴袋给袋机：需判断底部填充 vs 顶部填充，确保预制吸嘴后还有足够充料位置

## 八、给袋机选型补充（B 培训）

- 真空+充气不一定必须 8 工位：6 工位可通过集成模块实现，取决于真空/充气要求、大型设备配合、工位空间是否容纳新增设备
- 袋装载重：常规不触及设备极限，超重才专项评估；BS 套袋机型不涉及
- 实际产能：理论速度对得上公司机型；主要影响因素是**填料速度**（袋口越大物料更易进入，理论速度越高）；不简单用 40-70% 通用公式

## 九、跨机型计量通用规则（B 培训）

- 螺杆秤适用范围：粉末 + 细小颗粒 / 体积密度相对稳定的物料（不仅限粉末）
- 计量液体泵 / 多头组合秤：用在立式机和给袋机上的参数基本一样，不区分机型专用，只区分不同物料用不同计量工具
- 量杯计量：高油脂物料（花生）暂未反馈粘连漂移问题，后续工程师反馈再补充
- 带壳 vs 去壳坚果：主要影响是**物料体积**而非硬度，不带壳后变易碎则按易碎规则另议

## 十、易混术语对照（口语 vs 型号）

| 口语 | 含义 | 对应型号前缀 |
|---|---|---|
| 立式机 | 立式制袋（无工位、膜袋成型）| VP-FB-* |
| 给袋机 | 带工位（多工位 = 预制袋）| VP-BF-* |
| 工位 | 包装机的工序位数（如 6 工位 = 6 个工序位）| 型号末尾 -06 / -08 / -10 等 |

注意：旧章节五标题"立式给袋机"为历史培训用词，**口语准确说法是"给袋机"**。

**判定铁律（2026-06-17 B 强调）**：以 B 口语为准，不根据型号号猜。
- B 说"立式机" → 默认 VP-FB 无工位
- B 说"给袋机"或带"工位/N 位" → VP-BF
- 型号号（如 "420"）不足以推断机型，立式机和给袋机都有同号段
