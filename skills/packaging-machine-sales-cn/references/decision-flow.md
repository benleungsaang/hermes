# Decision Flow — One-Page Cheat Sheet

```
客户需求
  │
  ▼
[1] 物料形态?
  │
  ├─ 散料(粉/液/膏/颗粒)────┐
  ├─ 定型单品(面包/五金)───→ HFFS 枕式
  └─ 大件 + 预制袋装不下 ──→ 套袋 (横式给袋, 推入式)
                           │
  散料 / 散装颗粒 ──────────┤
                           ▼
[2] 袋型?  ←  此步 override 大类
  │
  ├─ 卷膜袋 (bag)  ────→ VFFS 立式 (VP-FB / VP-BS)
  ├─ 预制袋 (pouch) ──→ Premade Bag 给袋机 (VP-BF / HP-BS)
  └─ 枕式袋 ─────────→ HFFS 枕式 (HP-FB / HP-FB-BF)

                           ▼
[3] 膜宽/袋宽核算
  - VFFS / HFFS: 膜宽 = (成品袋宽 + 10~15mm) × 2
    ⚠️ 中封余量 ≥ 10mm; 接近极限膜宽时沟通选更大机型
  - Premade Bag: 按袋宽为主, 袋长为辅

                           ▼
[4] 枕式机额外确认 (仅 HFFS)
  - 走膜: 上走膜快 / 下走膜平稳 (易滚动 → 下走膜)
  - 切刀: 旋转(R)快/限矮 / 往复(B)高开口首选 / 上下切便宜/必停机
  - 高产品 + 高速是矛盾; 看旋转刀的两个中心距能否容下

                           ▼
[5] 给袋机工位 (仅 Premade Bag)
  - 6 工位: 标准 (拉链/开底袋基础增值也能完成)
  - 8+ 工位: 真空/充气/双落料/拉链预封
  - 真空+充气不一定必须 8 工位; 6 工位 + 集成模块可行

                           ▼
[6] 计量选型 (任一机型都查)
  - 粉末 → 螺杆秤 / 量杯
  - 易吸潮 / 压缩比易变物料 → ❌ 量杯 ❌ 螺杆秤 → 净重电子秤
  - 易碎颗粒 / 高精度 → 多头组合秤 (防碎定制版)
  - 液体/膏体 → 计量液体泵

                           ▼
[7] 配件清单 (报价必备)
  - 必配: 提升机 (VFFS / Premade Bag); 防碎秤 (易碎物料)
  - 可选: 打码机 / 视觉检测 / 输送机 / 料仓
  - 工艺升级: 抽真空 / 充氮气 / 排空气 / 开底袋 / 开拉链 / 封拉链 / 双落料

                           ▼
[8] 填装检查
  - 客户手工测试能装 ≠ 机器能装
  - 预计装到 70-80% 满 → 预警, 让客户重新评估袋型
```

## 速记

- 看袋型定机型, 不看物料定机型
- 散装颗粒 + pouch = 给袋机 (不要默认枕式)
- 真空充气 ≠ 必须 8 工位 (看具体要求)
- 计量选型优先看物料特性, 再看价格
- 报价 = 主机 + 必配件 + 选配件 + 工艺升级
